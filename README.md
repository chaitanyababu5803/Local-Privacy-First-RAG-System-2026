1. The "Zero-Cost" AI Power Projects

Project A: Local "Privacy-First" RAG System
The Concept: A secure document chat for banks/lawyers where data never leaves the building.
How to build for FREE:
LLM: Use Ollama (Free, runs Llama 3 locally on your RAM).
Backend: Java 21 + Spring AI (The enterprise standard in India).
Vector DB: ChromaDB or Milvus (Open-source, runs in a Docker container).
Frontend: React hosted on Vercel (Free tier).
Why HR loves it: It shows you can build AI without blowing the company's cloud budget.
